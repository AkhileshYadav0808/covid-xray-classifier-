 Detection of COVID-19 from Chest X-Ray

## 📌 Overview

This project aims to detect **COVID-19** from **Chest X-ray (CXR) images** using Deep Learning and Convolutional Neural Networks (CNNs). The model is trained to classify chest X-ray images into different categories such as **COVID-19**, **Normal**, and optionally **Pneumonia**.

The goal of this project is to assist healthcare professionals by providing a fast and automated screening tool based on chest X-ray images.

---

## 🚀 Features

- Detect COVID-19 from Chest X-ray images
- Deep Learning-based image classification
- Image preprocessing and augmentation
- Model training and evaluation
- Prediction on custom X-ray images
- Performance visualization using accuracy and loss graphs
